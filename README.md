Проект аналитика вакансий на сайтах HeadHunter и SuperJob.

Добро пожаловать в проект, который:
    1. Позволяет делать запросы по API к сайтам HeadHunter и  SuperJob, по названию вакансии.
    2. Обрабатывает получаенные данные по запросу.
    3. Выводить пользователю информацию о полученных вакансиях.
    4. Поддерживает сравнение вакансий по заработной плате.
    5. Поддерживает фильтрацию вакансий по заработной плате и городу.
    6. Сохраняет информацию о вакансиях в Json файлы.
    7. Удаляет вакансии по id.

[parsing.py](src%2Fparsing.py) - создержит классы для отправки запроса к HeadHunter и SuperJob и обработки полученных данных
[vacancy.py](src%2Fvacancy.py) - содержит класс в котором реализованы методы сравнения вакансий по зарплате
[save_to_file.py](src%2Fsave_to_file.py) - содержит классы для добавления вакансий в файл, сохранения список вакансий в файл, удаления вакансий и получение из файла списка вакансий по фильтру заработная плата и город.
[userinteraction.py](src%2Fuserinteraction.py) - класс для взаимодействия с пользователем.
